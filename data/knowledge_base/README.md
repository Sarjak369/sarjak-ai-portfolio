# Knowledge Base

This directory contains the knowledge base documents for the AI portfolio assistant.

## Setup Instructions

To use this portfolio:

1. **Fill in your information** in the following markdown files:
   - `about_me.md` - Personal information, visa status, hobbies
   - `contact.md` - Contact information and preferences
   - `education.md` - Academic background and achievements
   - `experience.md` - Work experience and accomplishments
   - `projects.md` - Featured projects and GitHub links
   - `publications.md` - Research papers and publications
   - `skills.md` - Technical skills and competencies

2. **Format**: Use markdown format with clear sections and headers

3. **Content Guidelines**:
   - Be specific and detailed
   - Include metrics and achievements
   - Add relevant links (LinkedIn, GitHub, etc.)
   - Keep information professional and up-to-date

## File Templates

Each file should follow this general structure:
```markdown
# [Section Title]

## [Subsection]
- Point 1
- Point 2

## [Another Subsection]
Content here...
```

## Privacy Note

These files are gitignored by default. Add your personal information locally and the RAG system will index them automatically.
